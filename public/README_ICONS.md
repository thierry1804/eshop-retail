# Icônes PWA

Pour que l'application PWA fonctionne correctement, vous devez créer les icônes suivantes dans ce dossier :

- `pwa-192x192.png` - Icône 192x192 pixels
- `pwa-512x512.png` - Icône 512x512 pixels
- `favicon.ico` - Favicon standard
- `apple-touch-icon.png` - Icône pour iOS (180x180 pixels)

## Génération des icônes

Vous pouvez utiliser un outil en ligne comme :
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

Ou créer manuellement les icônes avec un outil de design (Figma, Photoshop, etc.).

Pour l'instant, l'application fonctionnera sans ces icônes, mais elles amélioreront l'expérience utilisateur lors de l'installation sur l'écran d'accueil.
